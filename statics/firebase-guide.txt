Firebase Rule 
---------------------------------------------------------------------------------------------
Note: If you have created your own firestore database.
Just paste the below given rule in the firestore > Rules and click Publish to apply rule.

Paste inside match /databases/{database}/documents { here... } :

match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
    	allow read, write:  if request.auth.uid == userId;
    }
}

Steps for hosting :

1. Install firebase CLI:
npm install -g firebase-tools

2. Sign in to Google, hit the below command and sign in:
firebase login

3. Initiate your project, Run this command from your app's root directory:
firebase init

4. Deploy to Firebase Hosting
When you're ready, deploy your web app Put your static files (e.g., HTML, CSS, JS) in your app's 
deploy directory (the default is "public"). Then, run this command from your app's root directory:
firebase deploy
